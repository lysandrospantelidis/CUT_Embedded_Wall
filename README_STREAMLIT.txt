CUT Embedded Wall Streamlit v7.4

How to run on Windows:
1. Extract this ZIP.
2. Double-click run_streamlit.bat

Or from PowerShell in this folder:
python -m pip install -r requirements_streamlit.txt
python -m streamlit run streamlit_app.py

Main file:
streamlit_app.py

The solver modules, assets, manual.pdf and CUT_Embedded_Wall_Description.pdf are included.
